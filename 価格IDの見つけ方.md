# Stripe価格ID（Price ID）の見つけ方

## 📋 価格IDとは？

価格ID（Price ID）は、製品（Product）に紐づく**価格情報**を表すIDです。
- 形式: `price_...`で始まる文字列
- 製品ID（`prod_...`）とは別物です

---

## 🔍 価格IDを確認する方法

### 方法1: 製品の詳細ページから確認

1. Stripeダッシュボードにログイン: https://dashboard.stripe.com
2. **「テストモード」がON**になっていることを確認（右上のスイッチ）
3. 左メニューから「**製品**」をクリック
4. 製品一覧から、作成した製品をクリック
5. 製品の詳細ページが開きます

**価格IDの表示場所**:
- 製品の詳細ページの「**価格**」セクションに表示されます
- 価格が既に作成されている場合、価格ID（`price_...`）が表示されます
- 価格が複数ある場合は、各価格の横に価格IDが表示されます

### 方法2: 価格一覧から確認

1. Stripeダッシュボードで「**製品**」→「**価格**」をクリック
2. 価格一覧が表示されます
3. 各価格の行に価格ID（`price_...`）が表示されます
4. または、価格をクリックして詳細ページで確認

---

## ⚠️ 価格IDが見つからない場合

価格IDが見つからない場合、**価格がまだ作成されていない**可能性があります。

製品（Product）を作成しただけでは、価格IDは生成されません。
**価格（Price）を作成する必要があります。**

---

## 🛠️ 価格を作成する手順

### 手順1: 製品の詳細ページから価格を追加

1. Stripeダッシュボードで「**製品**」をクリック
2. 製品一覧から、作成した製品をクリック
3. 製品の詳細ページで「**価格を追加**」または「**Add price**」ボタンをクリック

### 手順2: 価格の設定

以下の設定で価格を作成してください：

| 項目 | 設定値 |
|------|--------|
| **価格タイプ** | 「**定期**」（Recurring / Subscription）を選択 |
| **価格** | `200` |
| **通貨** | `JPY`（日本円）を選択 |
| **請求期間** | 「**月次**」（Monthly）を選択 |
| **請求間隔** | `1`（1ヶ月ごと） |
| その他の設定 | デフォルトのままでOK |

### 手順3: 価格を保存

1. 「**価格を追加**」または「**Add price**」ボタンをクリック
2. 価格が作成されると、価格ID（`price_...`）が表示されます
3. 価格IDをコピーしてください

---

## 📸 画面の見つけ方（日本語UIの場合）

### 製品の詳細ページで価格を確認

```
製品名: [あなたの製品名]
├─ 製品ID: prod_...
└─ 価格セクション
   └─ 価格ID: price_... ← ここに表示されます
```

### 価格一覧で確認

```
製品 > 価格
┌─────────────────────────────────────┐
│ 価格ID          | 金額  | 請求期間  │
├─────────────────────────────────────┤
│ price_1ABC...   | ¥200  | 月次     │ ← ここに表示
└─────────────────────────────────────┘
```

---

## 🔄 価格を直接作成する方法（製品がない場合）

もし製品がまだない場合は、価格を作成すると自動的に製品も作成されます：

1. Stripeダッシュボードで「**製品**」→「**価格を追加**」をクリック
2. 価格の設定を行います（上記の手順2を参照）
3. 価格を作成すると、価格IDが表示されます

---

## ✅ 価格IDを取得したら

価格IDを取得したら、`.env.local`ファイルに設定してください：

```env
STRIPE_PRICE_ID_GROW=price_1ABC123def456GHI789
```

（`price_1ABC123def456GHI789`は実際の価格IDに置き換えてください）

**重要**: 
- コメント（`#`）を外してください
- 製品ID（`prod_...`）ではなく、価格ID（`price_...`）を設定してください

---

## 🆘 それでも見つからない場合

### 確認事項

1. **テストモード/本番モードの確認**
   - 右上のスイッチで「テストモード」がONになっているか確認
   - テストモードと本番モードでは、異なる価格IDが表示されます

2. **製品の確認**
   - 製品が正しく作成されているか確認
   - 製品の詳細ページを開いて、価格セクションを確認

3. **価格の作成**
   - 価格が作成されていない場合は、上記の手順で価格を作成してください
